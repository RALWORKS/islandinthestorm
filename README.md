# ISLAND IN THE STORM
Competition entry for IFComp 2019

Proof of concept for IntFicPy engine.
Engine is bundled, frozen at competition release.

# Setup
1. install pipenv
https://pypi.org/project/pipenv/

2. start pipenv

```
pipenv shell
```

3. install dependencies

```
pipenv install
```

# Run Game

From inside pipenv ...

```
python islandgame.py
```
